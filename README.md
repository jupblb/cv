# My CV

After many years of using [moderncv] I decided to take a simpler approach. I've
been using Pandoc recently a lot and it has a great support for using [markdown]
as language for various document types.

## How to generate?

``` sh
# Default choice
$ make cv.pdf
```

  [moderncv]: https://github.com/xdanaux/moderncv
  [markdown]: https://pandoc.org/MANUAL.html#pandocs-markdown
